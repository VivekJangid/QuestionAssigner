# QuestionAssigner
Simple application to add assignments and select and send individual assignment to candidate's mail

## Requirements 

```bash
You need to have Python3 pipenv and npm installed.

  pip3 install pipenv
  npm i
```

## Running Instructions

Setting Up Server

```bash
git clone https://github.com/VivekJangid/QuestionAssigner.git
cd QuestionAssigner
pipenv shell
pipenv install
```

Make DB migrations
```bash
python manage.py makemigrations
python manage.py migrate
```

Run Server
```bash
python manage.py runserver
```

 
 Setting up React
 ```bash
 Open New Tab on Terminal
 cd frontend
 npm i npm
 ```
 
 Running React
 ```bash
 npm run dev
 ```
 
Open Browser
```bash
Goto   localhost:8000
```
